= phones-brazil

Validates Brazilian phone numbers. Currently works only for cellphones.

Validates area code (2 digits and existing area codes) and cellphone number size (currently 9 digits for area codes 11-28)

== Usage

  class Phone < ActiveRecord::Base
    validates_with PhonesBrazil::Validator::CellPhoneValidator
  end
